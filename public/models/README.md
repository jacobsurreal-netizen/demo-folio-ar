# Models

Place artifact GLB files here.

- `artifact.glb` — tetrahedron-orb artifact entity (strong silhouette, emissive materials)

Requirements:
- GLB format (binary glTF, single file)
- ≤ 50K triangles recommended for mobile AR
- MeshBasicMaterial or baked emissive textures preferred
- No PBR, no environment maps
